# Quantum Tour (client)

    The client project for QGame "Quantum Tour"

----

### install

- install [Miniconda](https://docs.conda.io/projects/miniconda/en/latest/)
  - `conda create -n isq python==3.8`
  - `conda activate isq`
  - `pip install -r requirements.txt`


### start

- run `run.cmd`
